Mandelbrodt Convergency Radius

Math is beautiful! I really like mathematical theory, especially if it is as beautiful as the mandelbrodt set. On the horizontal axis is the real part and on the vertical axis is the imaginary part of the complex number z_i = z_{i+1}+c. If we iterate this complex number sequence until the convergency radius is small enought and print this result as a color based on the convergency radius, we see the full beauty of mathmatical equations. Do you like math too? Try out the code of my project. Its very simple to run the docker container and produce this image and try out other complex number sequences and other parts of the mandelbrodt set. The project is written in plain C++ with no libraries. The idea for this project came from a university assignment that I did during my masters studies.

Dijkstras Shortest Path Algorithm

Dijkstras shortest path algorithm is widely used in appilcations from autonomous driving over game development to financial decision making. The project that you'll find on github is a just started try of implementing my own version of it from scratch. The focus was not to make the prettiest code or the most convenient implementation, but to get a very basic hands on to graph algorithms, especially to learn about the two implementation possabilities of implementing it as a graph class or a matrix. Have fun looking at this currently unfinished project.
